# Product Gallery — CSS Grid

A modern product gallery built using **HTML5** and **CSS Grid** as part of my Full Stack Web Development Journey.

## Features

- Responsive-style product grid layout
- Staggered card sizing using `grid-column: span`
- Product image, name, price, and Buy Now button
- Hover effects and smooth transitions
- Clean card design with shadow and rounded corners

## Technologies Used

- HTML5
- CSS3 (Grid Layout)

## Concepts Practiced

- display: grid
- grid-template-columns
- repeat()
- fr unit
- grid-column: span
- gap
- Hover effects
- Box Shadow
- Border Radius
